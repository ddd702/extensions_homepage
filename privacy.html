Privacy Policy

Effective Date: July 10, 2026

ScreenCapture is committed to protecting user privacy. This extension adheres to a strict privacy-first architecture:

  • No Data Collection: We do NOT collect, store, or transmit any personal data, browsing history, or website content to external servers.
  • Local Storage: All captured images and history metadata are processed and stored strictly within the user's local browser database (IndexedDB) via unlimitedStorage.
  • On-Demand Permissions: Permissions like activeTab and scripting are requested solely to measure dimensions and execute scrolling actions on-demand when you click the extension.
  • Optional Host Permissions (<all_urls>): The optional_host_permissions for all URLs is strictly disabled by default and entirely optional. It is only requested if the user chooses to enable it to ensure feature persistence and workflow continuity (e.g., keeping the side panel workspace functional and uninterrupted during page refreshes, domain redirects, or cross-domain navigation).
